# Log4Shell Docker Lab Helpers

This directory contains Log4Shell-era scanner and helper material.

`requirements.lab.txt` is intentionally named so this research lab is not
mistaken for a maintained Python project by dependency tooling.

To run the helper intentionally, copy or reference the lab requirements file:

```bash
python3 -m pip install -r requirements.lab.txt
```

Do not deploy this material to shared environments without review.
