# MCP Runbooks

This directory holds operational playbooks and long-form working drafts for MCP
red-team, blue-team, and incident-response workflows.

## Files

- `beta__.md`: Red-vs-blue MCP security runbook covering multi-agent mesh threat
  models, detection engineering, incident response, hardening, regression, and
  capability development.

## Handling Notes

Runbooks may contain generalized offensive scenarios. Keep target-specific
details and evidence out of this public archive unless intentionally sanitized.
