# MCP Tooling Notes

This directory tracks MCP scanner, guardrail, and supporting tool research.

## Files

- `scanner-landscape.md`: Practitioner-first index of MCP scanners, guardrails,
  drift detection tools, and how they map to a zero-trust agent architecture.

## Use

Use this area for tool landscape notes, selection criteria, scanner comparisons,
and operational fit. Keep runnable prototype code in a dedicated subdirectory
once it becomes more than a note.
