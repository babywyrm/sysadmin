# MCP Assessment Methodology

This directory contains MCP security assessment frameworks and pentest notes.

## Files

- `security-assessment-framework.md`: Practitioner risk-card format for MCP and
  agent tool security testing.
- `security-assessment-framework-v3.md`: Expanded assessment matrix mapped to
  MCP/agent risk categories and blue-team signals.

## Use

Use these files when planning MCP red-team, purple-team, or design-review work.
Keep concrete target notes out of this public research archive unless they are
sanitized and intentionally general.
