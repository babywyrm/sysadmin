# MCP Arbiter Prototypes

This directory contains larger Python prototype scanners/arbiter tools for MCP
and DVMCP-style security assessment.

## Files

- `prime.py`: Hardened universal MCP/DVMCP scanner prototype with target
  discovery, attack-chain correlation, and reporting logic.
- `beta__.py`: Earlier or alternate scanner/arbiter prototype retained for
  comparison before merging or pruning.

## Handling Notes

These are research prototypes, not packaged tools. Review dependencies, target
handling, and output behavior before running against any environment.
