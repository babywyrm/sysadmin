# MCP Architecture Notes

This directory contains architecture sketches for secure MCP and agent-to-tool
execution.

## Files

- `security-architecture-v2.md`: Layered MCP security architecture and control
  model.
- `zero-trust-tool-execution.md`: Vendor-neutral zero-trust tool execution and
  session flow design.
- `aws-agentic-mesh.md`: AWS-native agentic mesh design using IAM, STS, Bedrock,
  and AWS-native policy/audit services.

## Use

Start here when designing a new MCP deployment, agent gateway, or tool execution
environment.
