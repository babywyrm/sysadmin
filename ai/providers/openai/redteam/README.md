# OpenAI Red-Team Notes

This directory contains provider-era prompt testing and adversarial prompt
engineering notes.

## Files

- `prompts__.md`: Prompt collections and testing notes.
- `prompts_runbook_.md`: Adversarial prompt engineering workflow and response
  interpretation runbook.
- `tester__.py`: Prompt testing script or prototype retained for review.

## Handling Notes

Keep target-specific prompts, outputs, and evidence sanitized before publishing.
Promote reusable methodology into `../../mcp/redteam/` or `../../llms/` when it
becomes vendor-neutral.
