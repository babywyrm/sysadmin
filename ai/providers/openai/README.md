# OpenAI Provider Experiments

This directory contains provider-specific OpenAI-era examples and notes. Much of
the code is historical and may use deprecated completion APIs.

## Layout

- `claude/`: Claude comparison and dynamic prompt experiments kept with the
  OpenAI provider notes because they were originally captured together.
- `legacy-demos/`: Early OpenAI API demos, Flask examples, and support files.
- `mcp/`: Provider-specific MCP notes, STRIDE material, and policy sketches.
- `redteam/`: Prompt-testing and adversarial prompt engineering notes.

## Usage Notes

Treat this as an examples archive. Do not assume examples are production-ready or
compatible with current provider SDKs without reviewing the API version first.
