# Provider-Specific MCP Notes

This directory contains MCP notes and sketches that were captured with
OpenAI/provider experiments.

## Files

- `STRIDE.md`: STRIDE threat model for MCP as a Kubernetes control plane.
- `brief__.md`: Security overview for Kubernetes-native MCP controller
  execution.
- `labs__.md`: Lab or experiment notes for provider/MCP work.
- `red_team_.md`: Provider-specific red-team notes.
- `stride___.py`: STRIDE-related script or prototype.
- `gatekeeper/`: Gatekeeper/policy demo material.

## Handling Notes

Vendor-neutral MCP material should live under `../../mcp/`. Keep files here only
when they are tied to provider-specific experiments or historical capture.
