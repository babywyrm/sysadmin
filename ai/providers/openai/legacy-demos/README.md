# OpenAI Legacy Demos

This directory preserves early OpenAI API demos and small Flask examples.

## Layout

- `code-generation-loop/`: Copied/public example of a loop that asks an OpenAI
  completion model to generate code, execute it, and iterate.
- `dinosaur-flask/`: Small Flask app that restricts prompts to dinosaur-related
  questions.
- `infosec-flask/`: Security/chatbot demo sketches with Flask and container
  notes.
- `pet-name-flask.py`, `pet-name-flask-static/`, `pet-name-flask-templates/`:
  OpenAI quickstart-style pet-name Flask example and support files.

## Maintenance Note

These examples may use deprecated SDKs or model names. Keep them as historical
research samples unless intentionally modernizing them in a separate pass.
