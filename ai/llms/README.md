# LLM Security Notes

This area contains vendor-neutral LLM security research, local model notes,
deployment sketches, and reference material.

## Layout

- `research/`: Literature reviews, paper lists, copied public references, and
  long-form reading notes.
- `agentgateway/`: Agent gateway Helm and Kubernetes deployment material.
- `bedrock/`: AWS Bedrock snippets and wrapper experiments that are specific to
  LLM runtime integration.
- `tooling/`: Small helper programs and experiments.
- `2024__.md`, `2024_modeling_.md`, `bedrock__.md`, `compare__.md`: Earlier
  working notes retained in place until they are reviewed individually.

## Handling Notes

Treat files in this area as research artifacts. Some examples are intentionally
old, copied from public references, or written against deprecated APIs. Prefer
adding context before rewriting them.
