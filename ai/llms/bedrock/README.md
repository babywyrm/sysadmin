# Bedrock Integration Snippets

This directory contains AWS Bedrock integration experiments and wrapper code.

## Files

- `bedrock_guardrails_.py`: Guardrail-related Bedrock experiment.
- `client_snippet_.py`: Minimal client usage snippet.
- `middle_.py`: Middleware or adapter experiment.
- `wrapper_.py`: Wrapper experiment for Bedrock interaction.

Related long-form notes live at `../bedrock__.md`.

## Handling Notes

These files are snippets, not a cohesive package. Check SDK versions and IAM
assumptions before reuse.
